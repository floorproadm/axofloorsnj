Fluxo de Medição e Precificação do SPU

# Sistema de Precificação SPU - FloorPRO## 1. Configuração do Catálogo### 1.1 Serviço de Lixa como exemplo - **Hardwood Floor Sanding**    - Pre-finish (taxa específica por sqft)    - New wood (taxa específica por sqft)    - Diferentes acabamentos/produtos- **Stairs**    - Preço por degrau    - Opção de preço por projeto completo    - Variações por tipo de escada- **Hand Rails**    - Preço por linear feet    - Opção de preço por projeto    - Adicionais (balusters, detalhes)### 1.2 Personalização de TaxasCada SPU pode configurar:- Taxas por sqft para cada tipo de piso- Preços unitários para escadas- Valores por linear feet para corrimãos- Taxas para serviços adicionais## 2. Processo de Medição### 2.1 Floor Areas```CopyÁrea Total = Comprimento × LarguraValor Piso = Área Total × Taxa por sqft```### 2.2 Stairs```CopyValor Escada = Número de Degraus × Preço por DegrauOUValor Escada = Preço Projeto Completo```### 2.3 Hand Rails```CopyValor Corrimão = Linear Feet × Preço por ft+ Adicionais (balusters, etc.)OUValor Corrimão = Preço Projeto Completo```## 3. Composição do Preço Final### 3.1 Custos Base```CopyCusto Material = Supply Orders TotalCusto Labor = Diária × Número de DiasCustos Adicionais = Extras + Complementos```### 3.2 Valor Total```CopyValor Total = Valor Piso + Valor Escada + Valor Corrimão + AdicionaisLucro Líquido = Valor Total - (Custo Material + Custo Labor + Custos Adicionais)```## 4. Exemplo Prático (Caso Rebecca)### Medições e Valores- Floor: 1500 sqft × $2.75 = $4,125- Stairs: 15 steps × $55 = $825- Rails: 100 ft × $35 = $3,500- Extras (balusters): $500- **Total Medição: $8,950**### Custos- Material: $1,000 (Supply Master)- Labor: 8 dias × $300 = $2,400- **Total Custos: $3,400**### Resultado Final- Valor Cobrado: $9,100- Custos Totais: $3,400- **Lucro Líquido: $4,790**## 5. Notas Importantes- SPU tem total autonomia para definir suas taxas- Preços podem variar por tipo de serviço- Sistema permite flexibilidade entre preço unitário ou por projeto- Adicionais podem ser incluídos conforme necessidade
